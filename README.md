# AI-Powered Sales Effectiveness Prediction

## Overview

This project develops an AI-powered Sales Effectiveness Prediction System for business lead analysis and prioritization.

The objective is to classify sales leads into High Potential and Low Potential categories using machine learning techniques, helping organizations focus on the most valuable opportunities and improve conversion rates.

## Business Problem

Sales teams often spend significant effort on leads that are unlikely to convert.

This project uses machine learning to:

* Predict lead quality
* Improve sales prioritization
* Support business decision-making
* Increase conversion efficiency
